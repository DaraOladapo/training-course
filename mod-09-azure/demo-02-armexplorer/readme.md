# Azure Resource Explorer

This demo shows the browser-based ARM explorer.

1. Navigate to https://resources.azure.com & login using an account that has access to an Azure subscription.
1. Click around showing how to see different things within the browser experience.
1. Highlight the following things:
  - Subscriptions
  - Resource groups
  - read & read/write mode
  - perform actions
  - get documentation on properties
  - equivalent PowerShell commands
